# Rostra-Backend
Use flask to provide restful api  for rostra app

## Requirements
- Flask
- [MongoDB](https://docs.mongodb.com/manual/installation/)
- Python 3.7

### Installed Packages

    pip install Flask
    pip install flask_mongoengine
    pip install flask-restx
    pip install flask-cors


## Start 

    flask run
